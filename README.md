# 🚀 Reto: 100 Proyectos con JavaScript Vanilla

¡Hola! Soy **DaniCodex**, estudiante de Ingeniería de Sistemas y creador de contenido. 
Este repositorio documenta mi progreso en el reto de construir **100 proyectos web** utilizando únicamente HTML, CSS y JavaScript puro (Vanilla JS), sin frameworks ni librerías.

El objetivo es dominar las bases de la web: manipulación del DOM, asincronía (APIs), manejo de estado, LocalStorage y lógica de negocio compleja.

## 🛠️ Tecnologías utilizadas
* HTML5 (Semántica y accesibilidad)
* CSS3 (Grid, Flexbox, Responsive Design)
* JavaScript (ES6+, Async/Await, Fetch API, DOM Manipulation)
